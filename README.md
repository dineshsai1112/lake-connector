## 🚀 What this app does

lake-connector helps you connect to a data lake and read files for machine learning work. It gives you a simple way to open data sets stored in cloud or local lake folders, then use them in your own workflow.

Use it when you want to:
- Read data lake files from one place
- Pull data into a machine learning task
- Work with common file formats used in data lakes
- Keep your setup simple on Windows

## 🖥️ What you need

Before you start, make sure your Windows PC has:
- Windows 10 or Windows 11
- An internet connection
- Enough free disk space for the app and your data files
- Permission to save files to your Downloads folder
- A file extractor if the download comes in a .zip file

For smooth use, it also helps to have:
- A recent version of Microsoft Edge or Google Chrome
- Basic access to the data lake location you want to read from
- A folder ready for your input files

## ❓ Troubleshooting

### The app does not open
Try these steps:
1. Right-click the file
2. Choose Run as administrator
3. Try again
4. Check that the download finished fully

### Windows blocks the file
Try these steps:
1. Right-click the file
2. Open Properties
3. Check for an Unblock option
4. Apply the change
5. Run the app again

### The connection fails
Try these steps:
1. Check the data path
2. Make sure the file or folder exists
3. Confirm the access details
4. Try a smaller test file first

### The data does not load correctly
Try these steps:
1. Check the file format
2. Open the file in a normal viewer to confirm it is not damaged
3. Re-download the file if needed
4. Make sure the source file uses the expected encoding
